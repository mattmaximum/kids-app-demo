# Road Trip I-Spy

A simple browser-based I-Spy game for kids ages 2–4, built for road trips.

## What it does

Tap **New Game** to get a randomly selected observation game with instructions. It cycles through 12 games and never repeats the last one. No install, no login, no internet required once loaded.

## The 12 games

| # | Game | Type |
|---|------|------|
| 1 | Color Spotter | Out the Window |
| 2 | Vehicle Hunt | Out the Window |
| 3 | Animal Lookout | Out the Window |
| 4 | Sign Shape Hunt | Out the Window |
| 5 | Sky Watch | Out the Window |
| 6 | Letter Hunt | Out the Window |
| 7 | Number Hunt | Out the Window |
| 8 | Touch It | In the Car |
| 9 | Body Part I-Spy | In the Car |
| 10 | Counting Game | In the Car |
| 11 | What's That Sound? | In the Car |
| 12 | Same Color as Me | Mixed |

## Tech

Single `index.html` file — plain HTML, CSS, and JavaScript. No frameworks, no build step, no dependencies.

Hosted on GitHub Pages: https://mattmaximum.github.io/kids-app-demo/
